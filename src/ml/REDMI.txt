=-=-= Пошаговая инструкция по запуску =-=-=
1. Входные тестовые данные подаются в файле "train.xlsx"
2. Запустите файл "processing_data_frame.py"
3. Запустите файл "model.py"
4. Запустите файл "load_model.py"
5. Запустите файл "server.py"
6. Откройте URL локального хоста: http://127.0.0.1:8080
7. Скопируйте тестовый запрос из файла "test_curl.txt"
8. Отправьте тестовый запрос в bash терминал
